# 福州

作为一个知识工作者，其首要不在于把事情做对，因凭着经验和知识，做对一件事情并不是一件难事。而做对的事情，贵在于把有限的时间放在首先需要解决的问题上。

道理总是相通的。
抓住主要矛盾，

问题并不出现在福州。


这么多年，闲言少语，


事在人为，


控制不住自己



到底还是迷了眼睛，
